# Stream URL Copy — Kodi Add-on

Add-on pro Kodi který umožnuje kopírování stream odkazů do programů jako VLC Media Player pro jejich stáhnutí.

## Jak stáhnout s VLC

1. Instaluj Add-on ze ZIPu.
2. Zapni libovolný stream a stopni ho.
3. Běž do hlavního menu, pak "doplňky" a pak klikni na "Stream URL Copy".
4. Otevři VLC Media Player, jdi do "Média" a "Převést / Uložit".
5. Vyber "Síť" a vlož kopírovaný odkaz, pak navol "Převést / Uložit".
6. Vyber profil "MP4", navol Cílový Soubor a stiskni "Spustit".
7. Buď trpělivý a pak si užiješ offline film. :D
